Order Sync

Shopee orders automatically create customer invoices in Odoo with correct product, quantity, and pricing.

💰

Payment Tracking

Shopee disbursements to SeaBank are recorded as journal entries. No manual reconciliation needed.

📊

Fee Accounting

Platform commissions, payment fees, and BigSeller charges are tracked as marketplace expenses.

↩️

Return Handling

Shopee returns and refunds are recorded against the correct COGS account automatically.

🔄

Daemon Mode

Run continuously with auto-sync every 30 minutes. Set and forget.

🔐

Secure

Direct Shopee API connection with HMAC-SHA256 signing. No data goes through third parties.
